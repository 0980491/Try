Introduccion

El presente proyecto consiste en la creación de un banco de objetos de aprendizaje, para facilitar la búsqueda de información académica, dirigida a los estudiantes del gimnasio nicolas de federman. entendiendo un objeto de aprendizaje como, una colección de elementos que se que están organizados en una dirección concreta, los cuales tienen la finalidad de enseñar. Y un banco como un espacio seguro, en el cual se pueden almacenar los distintos datos. esto para entregarle a los estudiantes una página web a la cual pueden acceder en cualquier momento, la página web estará dividida entre cursos y temas, esta página web se realizan actualizará periódicamente para poder agregar trabajos cada que sea necesario, la actualización se dará al momento de pedirle a los docentes de cada materia los trabajos que ellos consideren aptos para su publicación de cada tema visto a lo largo del periodo, esto se publicara y los estudiantes podrán tener acceso a esta información a partir de una página web. la página web se creará por medio del lenguaje de programación de python, y se publicará en un espacio en la nube de heroku.
